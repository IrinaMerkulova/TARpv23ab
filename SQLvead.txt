3.

VALUES(1, 'super lux');
Conversion failed when converting the varchar value 'super lux' to data type int.
-------------------
Данные записанны не в правильном порядке как в INSERT нужно в VALUES поменять местами

4.
delete table room_type;
Incorrect syntax near the keyword 'table'.
------------------
Чтобы удалить таблицу надо написать не delete а drop

5.
CREATE TABLE room_type(id int Primary key identity(1,1),
description varchar80,max_capacity int) ;
Column, parameter, or variable #2: Cannot find data type varchar80.
-------------------------------------
Недопустимый параметр, поскольку «varchar» не содержит круглых скобок. «varchar» не может быть без круглых скобок

id int Primary key identity(1,1),description varchar(80),
max_capacity int) ;SELECT  FROM room_type;
INSERT INTO room_type(description, max_capacity)VALUES ('super lux', 1),
 ('deluxe', 2),('super deluxe', 3),
('vip', 4), ('comfort', 5);
Incorrect syntax near the keyword 'FROM'.
----------------------------------------------
Неправильный синтаксис, так как после «Выбрать» и перед «FROM» должен быть знак «*», иначе он не будет работать.
