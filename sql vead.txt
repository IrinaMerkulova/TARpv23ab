kod:
CREATE TABLE Users (
    UserID INT PRIMARY KEY,
    UserName VARCHAR(50),
    UserEmail VARCHAR(100)
);

INSERT INTO Users (UserID, UserName, UserEmail) VALUES (1, 'Gleb', 'gleb@gmail.com');
INSERT INTO Users (UserID, UserName, UserEmail) VALUES (2, 'Bogdan', 'bogdan@gmailcom');
INSERT INTO Users (UserID, UserName, UserEmail) VALUES (3, 'Kirill', 'kirill@gmail.com');

SELECT * FROM Users WHERE UserName = 'Gleb' AND UserAge = 30;

Invalid column name 'UserAge'.


kuidas lahendada seda probleemi?:

6.
kod:
CREATE TABLE Orders (
    OrderID INT PRIMARY KEY,
    CustomerID INT,
    OrderDate DATE
);

INSERT INTO Orders (OrderID, CustomerID, OrderDate) VALUES (1, 101, '2024-04-17');
INSERT INTO Orders (OrderID, CustomerID, OrderDate) VALUES (2, 102, '2024-04-18');
INSERT INTO Orders (OrderID, CustomerID, OrderDate) VALUES (3, 103, '2024-04-19');

SELECT MAX(OrderDate), OrderID FROM Orders;


viga:
Column 'Orders.OrderID' is invalid in the select 
list because it is not contained in either an aggregate function or the GROUP BY clause.

kuidas lahendada seda probleemi?:
Добавьте столбец в оператор GROUP BY или используйте агрегатную функцию.
