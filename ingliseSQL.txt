SQL (Structured Query Language) - Язык структурированных запросов.
DDL - Язык определения данных
DML - Язык манипулирования данными
Database - База данных
Database management system - Система управления базами данных
Related table - Связанная таблица
Relational database - Реляционная база данных
Relationship - Связь
Record - Запись
Primary Key - Первичный ключ
Foreign Key - Внешний ключ
One-to-many relationship - Один-ко-многим отношением
One-to-one relationship - Один-к-одному отношением
Query - Запрос
Report - Отчет
Form - Форма
Data type / varchar, char, int, decimal, binary, varbinary, text, blob, enum, set - Типы данных / строка переменной длины, символьный, целый, десятичный, бинарный, переменная бинарная, текстовый, двоичный большой объект, перечисление, множество
