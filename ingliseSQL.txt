
SQL - Structured query language (SQL) is a programming language for storing and processing information in a relational database. 
Язык структурированных запросов (SQL) — это язык программирования для хранения и обработки информации в реляционной базе данных.

DDL - Data Definition Language (DDL) (Data Description Language) is a family of computer languages ​​used in computer programs to describe the structure of databases.
Язык определения данных (DDL) (язык описания данных) — это семейство компьютерных языков, используемое в компьютерных программах для описания структуры баз данных.

DML - Data Manipulation Language (DML) is a family of computer languages ​​used by computer programs or database users to retrieve, insert, delete, or change data in databases.
Data Manipulation Language (DML) (язык управления (манипулирования) данными) — это семейство компьютерных языков, используемых в компьютерных программах или пользователями баз данных для получения, вставки, удаления или изменения данных в базах данных. 

Database - an organized collection of structured information, or data, typically stored electronically in a computer system.
организованный набор структурированной информации или данных, обычно хранящихся в электронном виде в компьютерной системе.

Database management system - software to store, manage, retrieve, and manipulate data effectively.
программное обеспечение для эффективного хранения, управления, извлечения и манипулирования данными.

Related table - is a table function, and returns a table of values that contains all of the rows on the many-side that are related to the current row on the one-side.
это табличная функция, которая возвращает таблицу значений, содержащую все строки на многосторонней стороне, которые связаны с текущей строкой на одной стороне.

Relational database -  a type of database that organizes data into rows and columns, which collectively form a table where the data points are related to each other. 
тип базы данных, в которой данные организованы в строки и столбцы, которые вместе образуют таблицу, в которой точки данных связаны друг с другом.

Relationship -  the established associations between two or more tables. 
установленные ассоциации между двумя или более таблицами.

Record - A record type is a composite data type that consists of one or more identifiers and their corresponding data types.
Тип записи — это составной тип данных, состоящий из одного или нескольких идентификаторов и соответствующих им типов данных.
