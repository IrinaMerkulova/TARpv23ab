
SQL - Structured query language
DDL - Data Definition Language
DML - Data Manipulation Language
Database (База данных)- Структурированная коллекция данных, хранящихся в электронном виде.
Database management system (DBMS) (Система управления базами данных)- Программное обеспечение, облегчающее организацию, хранение, извлечение и защиту баз данных.
Related table (Связанная таблица)- Таблицы с установленными отношениями в базе данных.
Relational database (Реляционная база данных)- База данных, хранящая данные в таблицах с предопределенными отношениями.
Relationship (Отношение)- Связь между таблицами в базе данных.
Record (Запись)- Строка в таблице базы данных, содержащая связанные данные.
Primary Key (Первичный ключ)- Уникальный идентификатор записи в таблице.
Foreign Key (Внешний ключ)- Поле в таблице, ссылается на первичный ключ другой таблицы.
One-to-many relationship (Отношение один-ко-многим)- Отношение, при котором одна запись ссылается на несколько записей в другой таблице.
One-to-one relationship (Отношение один-к-одному)- Отношение, при котором каждая запись в одной таблице ссылается на ровно одну запись в другой таблице.
Query (Запрос)- Запрос данных из базы данных.
Report (Отчет)- Отформатированное представление данных из базы данных.
Form (Форма)- Элемент интерфейса для ввода или отображения данных.
Data type (Тип данных)- Определяет тип данных, которые могут храниться в столбце.
--------------------------
Constraints (Ограничения)- Правила, накладываемые на столбцы данных для обеспечения их целостности.
NOT NULL (Не NULL)- Гарантирует, что столбец не может содержать значения NULL.
UNIQUE (Уникальный)- Гарантирует, что все значения в столбце уникальны.
PRIMARY KEY (Первичный ключ)- Уникально идентифицирует каждую запись в таблице.
FOREIGN KEY (Внешний ключ)- Устанавливает отношения между таблицами.
CHECK (Проверка)- Обеспечивает выполнение определенного условия для значений, вводимых в столбец.
DEFAULT (По умолчанию)- Задает значение по умолчанию для столбца.
CREATE INDEX (Создать индекс)- Улучшает скорость извлечения данных, создавая индекс на таблице.
