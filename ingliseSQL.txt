Data type

VARCHAR (Variable Character): Строковый тип данных переменной длины. Может содержать строку переменной длины до определенного максимального размера.

CHAR (Character): Строковый тип данных фиксированной длины. Все значения в столбце заполняются пробелами до указанной длины.

INT (Integer): Целочисленный тип данных, представляющий целые числа без десятичных знаков.

DECIMAL: Точный числовой тип данных, который хранит числа с фиксированным числом знаков до и после десятичной точки.

BINARY: Тип данных, который хранит двоичные данные фиксированной длины.

VARBINARY (Variable Binary): Тип данных переменной длины для хранения двоичных данных.

TEXT: Тип данных для хранения больших текстовых значений переменной длины.

BLOB (Binary Large Object): Тип данных для хранения больших двоичных объектов, таких как изображения или аудиофайлы.

ENUM: Перечислимый тип данных, который позволяет задавать список возможных значений для столбца.

SET: Тип данных, который позволяет выбирать одно или несколько значений из заданного набора значений.

--------------------------

Constraints - Ограничения в базах данных - правила, устанавливающие условия на данные, например, что они не могут быть пустыми (NOT NULL) или должны быть уникальными
 
NOT NULL - в базах данных означает, что определённый столбец не может содержать пустые значения (NULL).
UNIQUE - то ключевое слово в языке структурированных запросов SQL, которое позволяет задать ограничение для столбца или группы столбцов таблицы таким образом, чтобы каждое значение в указанных столбцах было уникальным.

PRIMARY KEY (Первичный ключ): Уникальный столбец или набор столбцов в таблице, который идентифицирует каждую строку в таблице.

FOREIGN KEY (Внешний ключ): Столбец или набор столбцов, который устанавливает связь между двумя таблицами. Он ссылается на первичный ключ в другой таблице.

CHECK (Проверка): Условие, которое ограничивает значения, которые могут быть вставлены в столбец. Проверка гарантирует, что данные соответствуют определенным критериям.

DEFAULT (По умолчанию): Значение, которое будет установлено для столбца, если для этого столбца не указано другое значение при вставке новой строки.

CREATE INDEX (Создать индекс): Команда SQL, которая создает индекс для ускорения поиска данных в столбцах таблицы. Индекс упорядочивает данные в столбце для более эффективного доступа к ним.
