13.	create procedure tabeli_kustuta
	@tabeli_nimi varchar(50)
	AS
	BEGIN 
	DROP TABLE @tabeli_nimi
	END;
	EXEC tabeli_kustuta 'default_table'

	Msg 102, Level 15, State 1, Procedure tabeli_kustuta, Line 5 [Batch Start Line 20]
	Incorrect syntax near '@tabeli_nimi'.
		- Ошибка говорит о том , что программа ждет от нас того, чтобы мы написали название таблицы ввиде объекта,
		но получает название таблицы ввиде строчки, и программа не понимает.
		- Правильное решение такое :

		create procedure tabeli_kustuta
		@tabeli_nimi varchar(50)
		AS
		BEGIN 
		DECLARE @STMT AS VARCHAR(MAX);
		SET @STMT = CONCAT('DROP TABLE ', @tabeli_nimi);
		END;
		BEGIN;
		EXEC (@STMT);
		END;
		EXEC tabeli_kustuta 'default_table'
