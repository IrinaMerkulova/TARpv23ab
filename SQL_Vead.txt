select * from room

6.

insert into room_type(description,max_capacity)
values(1,'Martin');

Conversion failed when converting the varchar value 'Martin' to data type int.

lahendus:

Viga ilmneb, kui üritad lisada stringi 'Martin' arvulisse veergu. 
Lahenduseks kasuta stringi ümbritsetud ülakomadega ('Martin').

õige kood:

insert into room_type(description,max_capacity)
values('Martin',1);

8.

drop table room_type

Objekti 'room_type' ei saanud eemaldada, sest sellele viitab FOREIGN KEY piirang.
ei saa emaldada, sest room_type'ile on kinnitatud foreign key

11.

create procedure tabeli_kustuta
	@Martin_Sild varchar(50)
	AS
	BEGIN 
	DROP TABLE @MartinSild
	END;
	EXEC tabeli_kustuta 'Unustatud_tabel'

Incorrect syntax near '@Martin_Sild'.
Protseduur tabeli_kustuta
Vale süntaks '@Martin_Sild' juures.
Viga ütleb, et programm ootab, et me kirjutaksime tabeli nime objektina,
kuid ta saab tabeli nime stringina ja programm ei saa sellest aru.
lahendus

		create procedure tabeli_kustuta
		@Martin_Sild varchar(50)
		AS
		BEGIN 
		DECLARE @STMT AS VARCHAR(MAX);
		SET @STMT = CONCAT('DROP TABLE ', @Martin_Sild);
		END;
		BEGIN;
		EXEC (@STMT);
		END;
		EXEC tabeli_kustuta 'Unustatud_tabel'
